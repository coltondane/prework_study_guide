Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A CSS file can be added to an HTML file by providing the general path to the file after the meta and before the title
  • By calling HTML elements in CSS we can add CSS elements to them

Git

  • git status: checks what branch we are currently working on
  • git checkout -b 'branch-name': creates a new branch and switches to it
  • git add -A: stages the changes made
  • git commit -m "changes description": commits the changes to the repoitory
  • git pull origin main: syncs the local branch with the remote branch by pulling in the base branch
  • git push origin 'branch name': pushes all of the changes
  • update the local branch by running git chekout main and git pull origin main
